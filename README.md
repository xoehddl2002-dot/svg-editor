# SVG Editor

SVG 기반 이미지 편집기 개인 프로젝트입니다.
브라우저 환경에서 SVG 요소를 생성하고 선택·이동·크기 조절 등
기본적인 편집 동작을 구현했습니다.

SVG 요소를 DOM 구조로 관리하고
선택 상태와 속성 변경을 React 상태로 처리하는 방식으로
편집기 구조를 구성했습니다.

## 주요 기능
- SVG 요소 생성 및 선택
- 드래그 이동 및 크기 조절
- 속성 기반 편집 구조
- 선택 상태 관리

## 기술 스택
- React
- TypeScript
- Vite
- Tailwind CSS
- SVG DOM API

※ SVG Movie Editor 프로젝트의 기반이 되는 편집기입니다.
